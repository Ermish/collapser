## What is it?

Collapser is a jQuery plugin that lets you collapse any content and toggle its visibility.

## Usage:

It's really easy! 
1) Just wrap your contents with: ```<div class="collapser"></div>```
2) Initialize through JQuery:``` $('.collapser').collapser(); ```
